# SysFlow.ai

Real-time system design and AI-powered simulation platform.

## Architecture

- **Frontend**: Next.js
- **Backend**: Node.js microservices
- **Database**: PostgreSQL
- **Cache/Streams**: Redis
- **Orchestration**: Kubernetes
- **Cloud**: AWS EKS
- **AI**: OpenAI/Claude API

## Services

1. **api-gateway**: REST APIs + WebSocket server
2. **simulation-engine**: Discrete-event simulator
3. **metrics-service**: Aggregates and exposes metrics
4. **ai-worker**: AI-powered architecture analysis
